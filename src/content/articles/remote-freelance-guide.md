## 为什么选择远程接单？

远程工作已经成为程序员的常态选择。根据 Stack Overflow 2025 年调查，超过 58% 的开发者至少尝试过远程接单。在中国，随着数字经济的蓬勃发展，越来越多的企业愿意雇佣远程开发者。

**主要优势：**

- 🌍 **地点自由**：在家、咖啡馆、旅途中都能工作
- ⏰ **时间灵活**：自己安排工作时间，告别 996
- 💰 **收入上限高**：一线大厂薪资 + 副业接单，月入 3-5 万不是梦
- 📈 **技能成长**：接触不同项目，快速积累经验

## 国内接单平台对比

### 1. 程序员客栈 (proginn.com)

国内最专业的程序员远程工作平台。

- **适合人群**：有 2 年以上经验的开发者
- **项目类型**：Web 开发、小程序、App 开发
- **收入参考**：¥800-2000/天
- **优点**：项目质量高、有平台担保
- **缺点**：需要审核入驻

### 2. 猪八戒网 (zbj.com)

国内最大的综合性服务交易平台。

- **适合人群**：各阶段开发者
- **项目类型**：网站建设、软件开发、设计
- **收入参考**：¥500-5000/项目
- **优点**：项目量大、入驻门槛低
- **缺点**：竞争激烈、需要竞价

### 3. 码市 (mart.coding.net)

Coding.net 旗下的软件外包平台。

- **适合人群**：有经验的团队或个人
- **项目类型**：中大型软件项目
- **收入参考**：¥1万-10万/项目
- **优点**：项目规范、有项目管理工具
- **缺点**：偏向团队

## 国外平台推荐

### Upwork

全球最大的自由职业平台。

- **收入参考**：$20-100/小时
- **热门技能**：React、Python、Node.js、AI/ML
- **技巧**：初期可以低价接单积累好评，之后逐步提价

### Toptal

顶级自由开发者平台，只接受前 3% 的申请者。

- **收入参考**：$60-150/小时
- **要求**：需要通过严格的算法和英语面试
- **适合**：有 3 年以上经验的全栈/后端开发者

### Fiverr

从 $5 起步的服务市场，现在也有大量高单价项目。

- **热门服务**：Bug 修复、代码审查、WordPress 开发、API 集成
- **技巧**：创建多个不同价位的服务包，覆盖不同客户群体

## 定价策略

### 新手定价建议

| 经验等级 | 时薪范围（国内） | 时薪范围（国外） |
|---------|--------------|--------------|
| 入门（<1年） | ¥50-100 | $15-25 |
| 熟练（1-3年） | ¥100-200 | $25-50 |
| 资深（3-5年） | ¥200-400 | $50-100 |
| 专家（5年+） | ¥400-800 | $100-200 |

### 定价公式

```
合理时薪 = (全职月薪 × 1.5) / 160 小时
```

自由职业的成本包括社保、设备、学习时间，所以要在全职薪资基础上上浮 30-50%。

## 常见问题

**Q: 没有工作经验能接单吗？**

A: 可以从 Fiverr 的小单做起，或者先在 GitHub 上建立作品集。建议先用个人项目展示能力。

**Q: 英语不好能做国外平台吗？**

A: 可以先从国内平台做起。如果要接国外单，至少需要能读懂需求文档和进行基本沟通。

**Q: 接单和全职工作能同时做吗？**

A: 建议先全职工作 + 业余接单，等副业收入稳定超过主业后再考虑全职自由职业。
