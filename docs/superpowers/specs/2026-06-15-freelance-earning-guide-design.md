# Design: 自由职业赚钱指南网站

**Date**: 2026-06-15
**Status**: Approved
**Topic**: Freelance-earning-guide-website

---

## Overview

构建一个「自由职业赚钱指南」内容网站，面向所有阶段的自由职业学习者，通过免费内容引流 → 邮件订阅 → 数字产品 → 高端变现的四层漏斗实现站长自身盈利，同时帮助用户掌握不上班赚钱的技能。

## Target Audience

覆盖全阶段：零基础职场新人、有技能未变现者、已有自由职业想扩大收入者。

## Monetization Funnel

```
第1层：免费内容引流 (SEO + 社交媒体)
  ↓
第2层：邮件列表捕获 (订阅CTA + 免费诱饵 + 每周推送)
  ↓
第3层：数字产品销售 (电子书/模板/课程, ¥49-199)
  ↓
第4层：高端变现 (1对1咨询/付费社群/训练营, ¥499-2999)
```

MVP 阶段先实现第1层，后续逐步叠加。

## Site Structure (MVP)

```
🏠 首页        — 文章列表 + 分类导航 + 简介
📄 文章详情页   — 极简单栏阅读 + 目录 + 订阅CTA
📂 分类列表页   — 按分类筛选文章
ℹ️ 关于页      — 站长介绍 + 网站使命
🛠️ 工具推荐页  — 推荐工具合集（含联盟链接）
```

后续可扩展：课程页、社群页、用户系统、付费内容。

## Visual Design

- **风格**: 温暖活力风
- **配色**: 主色橙 #f97316, 浅橙 #fff7ed, 暖灰 #78716c, 白 #fff
- **首页**: 简洁标题 + 简介 + 文章分类标签 + 文章卡片列表
- **文章页**: 单栏居中，文章内容 + 底部作者信息 + 相关推荐 + 订阅CTA
- **导航**: 顶部固定，Logo + 文章 · 分类 · 工具 · 关于

## Tech Stack

- **框架**: Astro (静态站点生成器)
- **样式**: Tailwind CSS
- **内容**: Markdown / MDX
- **部署**: Vercel (免费, 自定义域名)
- **分析**: Umami 或 Plausible (隐私友好)
- **邮件**: ConvertKit 或 Buttondown (后续集成)
- **交互**: 后续按需引入 React/Vue 组件（Astro Islands）

## Design Decisions

1. **为何选择 Astro 而非 Next.js?** — Astro 默认零 JS，加载更快；Markdown 原生支持更好；代码量更少；对内容站而言是更优解
2. **为何选择单栏文章布局?** — 聚焦阅读体验，适合长文深度内容；移动端兼容性好
3. **为何先从内容引流开始?** — SEO 是长期资产；免费内容建立信任；验证需求后再投入付费产品
4. **为何用温暖配色?** — 降低新手心理门槛；与"自由""轻松"的品牌调性一致

## Non-Goals (MVP)

- 用户注册/登录系统
- 付费课程/支付集成
- 社区/论坛功能
- CMS 管理后台
- 国际化
