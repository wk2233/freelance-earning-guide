## ADDED Requirements

### Requirement: About page
The site SHALL have an About page introducing the site and its mission.

#### Scenario: Visitor reads about page
- **WHEN** a visitor navigates to `/about`
- **THEN** the page displays the site name "自由赚钱指南"
- **AND** the site mission: helping people earn money freely without traditional employment
- **AND** a brief introduction about the creator/curator

### Requirement: Tools recommendation page
The site SHALL have a Tools page listing recommended freelancing tools with external links.

#### Scenario: Visitor browses recommended tools
- **WHEN** a visitor navigates to `/tools`
- **THEN** the page displays categorized tool recommendations (e.g., 接单平台, 设计工具, 写作工具, 效率工具)
- **AND** each tool entry includes: name, brief description, and an external link
- **AND** the page may include affiliate/disclaimer note about links
